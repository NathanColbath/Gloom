# Keyboard shortcuts

Studio is still growing its shortcut map. Below is what is implemented today versus planned behavior.

## File and edit

| Action | Shortcut | Notes |
|--------|----------|-------|
| Save scene | *(menu only)* | **File → Save Scene**; Ctrl+S planned |
| Undo | *(menu only)* | **Edit → Undo** |
| Redo | *(menu only)* | **Edit → Redo** |

## Play mode

| Action | Shortcut | Notes |
|--------|----------|-------|
| Play / Stop | Toolbar buttons | No hotkey yet |

## Scene view

| Input | Action |
|-------|--------|
| Middle mouse drag | Pan editor camera |
| Scroll wheel | Zoom editor camera |
| Left click | Pick object (Hand or after pick with other tools) |

## Hierarchy

| Input | Action |
|-------|--------|
| Ctrl+click | Multi-select |
| Delete | Delete selected (with confirmation where applicable) |

## View menu

| Item | Effect |
|------|--------|
| Frame Scene | Fit scene bounds in Scene view |
| Reset Layout | Default dock split |

## Tips

- Focus **Game** view before testing WASD or mouse in scripts.
- Use **Edit → Undo** after gizmo drags in Scene view.
